Functions

Functions are reusable pieces of code. They can be named, like variables, and called when you need them.


            function chihuahuaFacts() {
              console.log("Chihuahuas are the smallest of all dogs.")
              console.log("Chihuahuas can be 'apple-headed' or 'deer-headed.'")
            }
Spend time explaining the syntax of a function. Run it in the console to show the difference between declaring and calling a function.

Functions with Arguments

Functions can accept one or more input values, called "arguments."


            function callDog(dogName){
              console.log("Come here, " + dogName  + "!");
            }

Function return values

Functions can return a value, allowing them to be used for common calculations.


                    function miles2Kilometers(miles) {
                        return miles * 1.60934;
                    }
                    console.log(miles2Kilometers(3));
                    // 4.82802

Multiple return values

Functions can have multiple return values. The first one to occur ends the function.


                    function letterGrade(score) {
                        if (score >= 90) {
                            return "A";
                        } else if (score >= 80) {
                            return "B";
                        } else if (score >= 70) {
                            return "C";
                        } else if (score >= 60) {
                            return "D";
                        }
                        return "F";
                    }

Arrays

Arrays are like lists for multiple values. Values are listed inside square brackets ( []) and separated by commas.


            var emptyArray = [];
            var arrayOfNumbers = [2, 4, 6, 8];
            var arrayOfStrings = ["cats", "dogs", "hamsters"];
            var arrayOfDifferentStuff = [23, null, "octopus", true];
            var arrayOfArrays = [arrayOfNumbers, arrayOfStrings];
Ask students to run this code themselves in their console. What would arrayOfArrays look like when eval'd?

Using Arrays

Find out how many values are in an array:


              var colors = ["red", "green", "blue"];
              colors.length

You can also retrieve values from an array:


              var firstColor = colors[0];
              var lastColor = colors[3];

Loops

A loop is a code structure that lets us repeat a task until a certain condition is met.

For loops are commonly used to count a certain number of times (iterations) to repeat a task.


            for ([initial value]; [condition]; [final expression]) {
              // do this thing
            }
          

            // count 1 to 10
            for (var i = 1; i <= 10; i++) {
              console.log(i);
            }

Spend time explaining what each part does. The concept of iteration is confusing at first!

It is useful to explain that i++ is equivalent to i = i + 1

Objects

An object in JavaScript is a more complex data structure that can store pairs of keys and values. It is a little like a dictionary.


            var anObject = {
              key: value,
              key: value
            };
          

            var aboutMe = {
              hometown: "New York, NY",
              almaMater: "UNC",
              likes: ["avocadoes", "chihuahuas", "code"],
              zodiacSign: "Scorpio"
            };
Demo examples in the console. Note comma-separated values, not semicolons.

Accessing Object Data

You can retrieve values using "dot notation"


              var aboutMe = {
                hometown: "New York, NY",
                almaMater: "UNC",
                likes: ["avocadoes", "chihuahuas", "code"],
                zodiacSign: "Scorpio"
              };
              var myHometown = aboutMe.hometown;

...or using "bracket notation" (like arrays)


              var mySign=aboutMe['zodiacSign'];
Demo accessing object data in the console.
