Overview: What is HTML?

HTML acts as the skeleton of a web page, providing its underlying structure. You can use HTML to specify parts of your page like:

  • header and footer
  • sections
  • text headings
  • body text
  • images
  • links
  • form inputs and buttons
  • items in a list

and more. HTML is also used to indicate what elements of a web page are logically enclosed in other elements. For example, a button that says "Submit Feedback" might belong inside a form, which is inside the same section as the heading, "Website Feedback".

CSS

Reminder: we saw this slide earlier in the overview. CSS provides style for the content we've built with HTML

HTML content has meaning and structure, but lacks presentation. Analogy to writing an essay: introduction/thesis statement, section headers, paragraphs, conclusion — and writing style is about how we present the content to the user (analogy to tone, word choice, how formal or informal your language is in an essay).

Awareness of audience is how you make these decisions — this is UX/UI: page design that takes into consideration how people will use and perceive our content.

As a web designer or developer, you need to put together your structure and content with consideration for what your audience needs and expects in order to get your point across most effectively.

Useful CSS Properties

  • color: the color of the text, by name or hex code
  • background-color: the color of the background, by name or hex code
  • background-image: an image to use for the background
  • font-size: the size of the text, in pixels (px) or em-units
  • text-decoration: none, underline, overline, line-through, blink
  • text-align: left, right, center, justify
  • height: the height of an element in px, %, or em-units
  • width: the width of an element in px, %, or em-units
